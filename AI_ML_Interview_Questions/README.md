# AI/ML Interview Coding Questions

This folder contains curated coding questions designed for **Gen AI/Agentic AI** and **AI/ML Engineer** interviews.

## Folder Structure

1. **01_python_fundamentals.py** - Core Python concepts and data manipulation
2. **02_data_structures_algorithms.py** - DSA questions relevant to ML systems
3. **03_nlp_llm_fundamentals.py** - NLP, text processing, and LLM concepts
4. **04_rag_vector_database.py** - Retrieval Augmented Generation patterns
5. **05_api_integration.py** - REST API design and integration
6. **06_prompt_engineering.py** - Prompt engineering and LLM interactions
7. **07_multi_agent_systems.py** - Agent-based and multi-agent architectures

## Interview Focus Areas

### For Job 1: Gen AI/Agentic AI - Python, RAG, Prompt Engineering
- Focus on: Files 3, 4, 6, 7
- Key frameworks: LangChain, LlamaIndex, CrewAI concepts
- Key skills: Prompt engineering, RAG, agent workflows

### For Job 2: AI/ML Engineer - AWS Bedrock, Python
- Focus on: Files 1, 2, 3, 5, 7
- Key services: AWS Bedrock, REST APIs
- Key skills: Python fundamentals, system integration, multi-agent workflows

## Difficulty Levels
- ⭐ Easy: Basic concepts, suitable for junior roles
- ⭐⭐ Medium: Intermediate concepts, most common in interviews
- ⭐⭐⭐ Hard: Advanced concepts, for senior/specialist roles

1. Start with Python fundamentals
